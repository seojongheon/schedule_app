# 오늘 할 일 시간 겹침 필터 설계

## 목표

`오늘 할 일`과 대시보드의 동일 목록에는 현재 사용자에게 배정되었고, 오늘의 시간 구간과 하나라도 겹치는 일정만 표시한다.

## 표시 규칙

- 오늘의 범위는 사용자 로컬 시간 기준 `오늘 00:00` 이상 `내일 00:00` 미만이다.
- 일정은 `startAt < 내일 00:00` 그리고 `endAt > 오늘 00:00`일 때 오늘과 겹친다.
- 따라서 어제 시작해 오늘 끝나는 일정과 오늘 시작해 내일 끝나는 일정은 모두 표시한다.
- 오늘과 겹치지 않는 과거 또는 미래 일정은 표시하지 않는다.
- 현재 사용자에게 배정되지 않은 일정은 기존과 같이 표시하지 않는다.
- 완료 여부와 일정 상태는 이 필터의 판단 기준에 추가하지 않는다.

## 구현 범위

- `ScheduleWorkspace`에 날짜 범위 겹침을 판단하는 순수 헬퍼를 둔다.
- `todaySchedules` 계산에 기존 참여자 필터와 시간 겹침 필터를 함께 적용한다.
- 대시보드와 `/dashboard/today`는 공통 `todaySchedules`를 사용하므로 동일한 결과를 표시한다.

## 검증

- 오늘 안에서 시작·종료하는 일정이 표시된다.
- 어제~오늘 및 오늘~내일 일정이 표시된다.
- 오늘과 겹치지 않는 일정과 미배정 일정은 표시되지 않는다.
- TypeScript 검사와 린트를 실행한다.
