# UI Contract: Weekly Calendar Preview

## Input

- 하루에 속한 일정의 순서 있는 목록
- 각 일정의 제목, 참가자 정보, 강조 여부
- 날짜의 오늘/선택 상태

## Output

- 입력 목록의 앞 2개를 일정 슬롯으로 표시한다.
- 나머지가 있으면 `+{hiddenCount}개` 형식의 더보기 슬롯 하나를 표시한다.
- 일정 슬롯과 더보기 슬롯은 각각 한 줄 높이를 유지한다.
- 슬롯은 해당 날짜의 7열 그리드 트랙보다 넓어지지 않는다.
- 긴 제목은 슬롯 경계에서 말줄임표로 생략한다.

## Interaction

- 날짜 셀 클릭은 기존 날짜 선택 동작을 실행한다.
- 더보기 슬롯에는 별도 클릭 또는 펼치기 동작을 만들지 않는다.
- 검색과 참가자 필터는 기존처럼 강조도만 바꾸며 슬롯 크기를 바꾸지 않는다.
- 주간 스와이프의 이전/현재/다음 패널은 같은 출력 계약을 사용한다.

## Out of Scope

- 월간 캘린더 일정 슬롯
- 선택 날짜 시간표의 일정 블록
- 일정 정렬, 저장, 권한, 색상 규칙
- 더보기 팝업 또는 목록 확장
